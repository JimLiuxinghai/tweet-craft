Notion Setup Guide

Connect Notion in minutes and reuse your existing database.

Setup Checklist

Takes ~6 minutes
01

Create Integration

Generate a Notion Integration Token (secret_/ntn_).

02

Share a Parent Page

Add the integration via Add connections on your page.

03

Connect in the Extension

Paste the token in Notion settings and connect.

04

Reuse or Create Database

Existing databases are reusable; otherwise create a new one.
